# CE Massage Website

## Overview

CE Massage is a modern and responsive website developed for a professional massage and wellness business. The goal of the project is to provide visitors with a clean, user-friendly platform where they can learn about available services, pricing, and the business itself.

The website focuses on delivering a professional online presence while ensuring a seamless experience across desktop, tablet, and mobile devices.

## Features

* Responsive design for all screen sizes
* Multi-language support (Hungarian and English)
* Service presentation with detailed descriptions
* Pricing overview
* Contact information and business details
* Modern and accessible user interface
* Mobile-friendly navigation with hamburger menu
* Optimized performance and user experience

## Technologies Used

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/CE_Massage_Frontend.git
```

Navigate to the project directory:

```bash
cd CE_Massage_Frontend
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
ng serve
```

Open:

```text
http://localhost:4200
```
